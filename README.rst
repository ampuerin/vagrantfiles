* Vagrantfiles

** Download and install Vagrant by Hashicorp
https://www.vagrantup.com/downloads

** Download and install Virtualbox
https://www.virtualbox.org

** Using Vagrantfiles
Start machine::
vagrant up

Update machine::
vagrant up --provision

Halt machine::
vagrant halt

Destroy machine::
vagrant destroy
